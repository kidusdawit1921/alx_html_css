
# Smiles School – Advanced HTML Project

## Overview

This is a **responsive and interactive HTML website** for the fictional **Smiles School** platform, designed to showcase courses, instructors, and tutorials in a modern, visually appealing layout. It demonstrates **semantic HTML, multimedia integration, and responsive design principles**.


## Features

* **Header & Navigation:** Logo and links to Course, Pricing, and Login pages.
* **Hero Section:** Catchy tagline with call-to-action button for free registration.
* **Instructor Showcase:** Profiles of smiling experts with images and awards.
* **Testimonials:** Blockquote section with user feedback.
* **Popular Tutorials:** Video thumbnails with titles, authors, ratings, and duration.
* **Free Membership Perks:** Highlights benefits with icons and descriptions.
* **FAQ Section:** Expandable answers for frequently asked questions.
* **Footer:** Logo and social media icons (Facebook, Twitter, Instagram).
